# 资本市场不同发展阶段下的风格因子有效性研究

本项目旨在解决的核心问题是：根据风格因子形成的指数更适合在何种阶段配置。

本研究通过研究A股市场中价值、成长、红利、低波、质量、规模、动量等风格因子在不同市场环境、行业属性和企业生命周期阶段下的表现差异，从指数化投资视角给出风格配置建议。

主要思路是用可观察、可投资、易解释的指数作为风格代理。

## 数据说明

原始数据来自iFinD手动下载，存放在`data/raw/ifind/`。日度行情字段包括开盘价、最高价、最低价和收盘价。

| 类别 | 代理指数 |
|---|---|
| 市场基准 | 沪深300 |
| 市场阶段划分 | 中证全指 |
| 风格因子 | 300价值、300成长、300SNLV、盈利质量、300动量、中证红利、中证500 |
| 行业属性 | 沪深300行业指数 |
| 企业生命周期 | 北证50、创业板50、中证A50、上国改革 |
| 利率环境 | 10年期国债收益率 |

数据处理口径：

- 日度交易价格取高开低收均价：`(开盘价+最高价+最低价+收盘价)/4`
- 周度观察点为每周三
- 若某指数周三没有有效价格，使用该指数此前最新一个交易日价格
- 指数尚未发布前保留为空
- 周度收益率相对沪深300计算超额收益

## 研究方法

### 1.市场环境

市场环境部分分析七个风格因子：价值、成长、红利、低波、质量、规模、动量。

市场阶段参照A股历史波动周期，并结合中证全指周度匹配均价寻找局部高点和低点，将样本划分为牛市、熊市和震荡市。随后计算各风格指数相对沪深300的年化超额收益、信息比率、胜率和最大回撤。

利率水平使用10年期国债收益率全样本分位数划分：

- 低于30%分位数：低利率
- 30%至70%分位数：中利率
- 高于70%分位数：高利率

波动环境使用中证全指26周年化波动率分位数划分：

- 低于30%分位数：低波动
- 30%至70%分位数：中波动
- 高于70%分位数：高波动

### 2.行业属性

行业属性部分在大盘股范围内分析六个风格因子：价值、成长、红利、低波、质量、动量。

以沪深300行业指数相对沪深300的周度超额收益为Y，以六类风格相对沪深300的周度超额收益为X，进行多元线性回归。回归系数用于衡量行业对风格暴露的敏感度，R方衡量风格模型对行业超额收益的解释度，残差衡量风格模型无法解释的行业波动。

同时使用高beta行业组合减低beta行业组合的收益差，评估某个风格能否在行业维度形成可操作的分层效果。

### 3.企业生命周期

企业生命周期部分同样分析六个风格因子：价值、成长、红利、低波、质量、动量。

| 生命周期阶段 | 代理指数 |
|---|---|
| 初创期 | 北证50 |
| 成长期 | 创业板50 |
| 成熟期 | 中证A50 |
| 调整期 | 上国改革 |

企业生命周期理论中也包括衰退期，但衰退期缺少稳定的可投资指数代理，因此未纳入实证分析。

本部分以生命周期代理指数相对沪深300的周度超额收益为Y，以六个风格因子相对沪深300的周度超额收益为X，观察不同生命周期代理对风格暴露的敏感度。

## 主要结论

- 熊市中，低波、价值和红利表现较好，成长和规模拖累较大。
- 震荡市中，规模和质量表现突出，价值和红利回撤较大。
- 牛市中，规模、质量和成长表现尚可，单一风格相对宽基的优势不突出，宽基指数本身具有较强配置价值。
- 中高波动环境下，质量和规模表现较强；低波动环境下，动量、价值和红利表现尚可。
- 中低利率环境下，质量和规模表现突出；高利率环境下，价值、红利和低波表现更好。
- 行业维度中，金融和地产更适合用价值框架观察，能源和原材料更适合用红利框架观察，医药、信息技术和主要消费更依赖成长、质量和动量。
- 生命周期维度中，初创期和成长期更依赖质量、成长等盈利弹性和基本面筛选，调整期更容易体现红利和现金流属性。

## 指数化投资建议

从指数化投资视角看，投资者可以先判断市场所处阶段，再选择单风格或复合风格指数：

- 风险偏好下行、市场进入熊市或利率处于高位时，优先配置红利低波、价值红利、低波动等防御型指数。
- 市场处于震荡市、利率中低位且盈利修复尚未全面扩散时，优先配置质量、质量成长、中证500等规模和基本面筛选型指数。
- 市场进入明确牛市后，以沪深300、中证A50等宽基指数作为核心仓位，同时用成长、动量或成长质量指数增强收益弹性。
- 单一因子指数适合表达清晰的阶段判断，复合因子指数适合作为长期配置工具，尤其是红利低波、质量成长、价值红利这类同时控制波动和收益来源的产品。

## 运行方式

```bash
pip install -r requirements.txt
```

